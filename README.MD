
# Laboratorio 3 - Gráficas por Computadora

En este laboratorio se renderizó una nueva figura a través del RayTracer simple que se ha estado trabajando en clase. Para lograr esto se investigó e implementó el Ray Intersect Algorithm de un triángulo.

En la imagen final generada por el programa se renderizó la figura tres veces con diferentes tamaños, posiciones y materiales (uno opaco, uno reflectivo y otro transparente) para demostrar el funcionamiento del programa.
